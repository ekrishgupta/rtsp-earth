# RTSP Earth (React Edition)

A Lofi/Monochrome 3D Globe visualization of RTSP streams, built with **React** and **react-globe.gl**.

![Preview](preview.png)

## Features
- **Lofi Aesthetic**: Minimalist white/black design with CRT scanlines.
- **3D Globe**: Interactive globe using `react-globe.gl` (Three.js).
- **Stream Visualization**: Red data points representing RTSP stream locations.
- **Live UI**: System status overlay with timestamps and coordinates.
- **Tech Stack**: Vite, React, react-globe.gl.

## Quick Start
1. Install: `npm install`
2. Run: `npm run dev`
3. Build: `npm run build`

## Controls
- **Rotate**: Left Click + Drag
- **Zoom**: Scroll Wheel
- **Pan**: Right Click + Drag
